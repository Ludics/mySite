终于搭建好了博客，发篇文纪念一下

Posted on 2016-04-20   |     |  

搞了一个下午终于搭建好了这个简单的博客，还是很开心的。

早就想做个自己的网站，可是一直都没弄，刚好前两天在简书上看到wingjay的一个教程，用Hexo和Github配合就能弄好了，原文在这。

wingjay的教程后半部分讲的有些简略，我又参考了其他一些博客才弄好。

大家搭好博客后都喜欢写一下搭的过程，但是我太懒了，而且反正这种教程多的是，所以就不写了。

不过还是要提一下，好多教程是针对Hexo 2.×的，但是现在已经更新到Hexo 3了，_config.yml的配置会有一些不同。

其实之前一直想用WordPress搭一个动态网站的，应该比这个静态的麻烦些。先这样吧，回头再试试WordPress。

我在这儿絮絮叨叨半天好像有人会看一样=_=

这两天再去买个域名，O(∩_∩)O哈哈哈~
